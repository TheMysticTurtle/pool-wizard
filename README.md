## ✨ What It Is

Pool Wizard is a lightweight pool chemistry calculator and reference guide for homeowners and pool professionals. Enter your test readings, get exact dose recommendations — no guessing, no cloud account, no nonsense.

It works fully offline. Everything stays on your phone.

## 🧪 What It Does

**The Calculator**
- FC, pH, total alkalinity, CYA, calcium hardness, combined chlorine, salt
- CYA-aware chlorine targeting (not just fixed 3–5 ppm)
- Algae & slime shock plans (green, pea soup, mustard, black, pink slime, biofilm)
- Ammonia breakpoint chlorination
- Phosphate remover dosing
- Step-by-step instructions with safety warnings and retest timing
- Trichlor / liquid / salt cell / manual sanitizer support

**The Spellbook**
- 9 chapters of pool chemistry reference — filtration, algae, stains, salt systems, problem water, and more
- Section search across all chapters
- Works fully offline — no internet needed poolside

**History & Profile**
- Save your pool profile once — gallons, surface type, sanitizer preference
- Log readings over time and see FC, pH, and alkalinity trends on a chart
- Everything stored locally on your device — nothing uploaded

## 📲 Install It

Pool Wizard is a **Progressive Web App** — it installs directly from your browser, no app store required.

**iPhone / iPad:** Open in Safari → Share ⬆ → Add to Home Screen

**Android:** Open in Chrome → Menu ⋮ → Add to Home Screen

**Desktop:** Click the install icon in Chrome or Edge's address bar

Once installed it launches full-screen like a native app and works with no internet connection.
